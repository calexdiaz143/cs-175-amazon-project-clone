About

This website uses a majority voting ensemble trained on 7,125 reviews from each category (171,000 in total).
It took approximately 48 hours to train this ensemble on the odin.ics.uci.edu server.
Tested on 2,375 reviews from each category (57,000 in total), the ensemble has an error rate of 16.7%.
The following table shows a breakdown of this ensemble.
Classifier Training Error Testing Error
Multinomial Naïve Bayes 0.194497 0.261053
Bernoulli Naïve Bayes 0.157228 0.245649
Logistic Regression 0.001936 0.214912
Random Forests 0.001450 0.229561
Gradient Boosting 0.184240 0.213105
